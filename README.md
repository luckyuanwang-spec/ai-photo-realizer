# AI Photo Realizer

**Make AI-Generated Images Look Like Real Photos — Free Online Tool**

One-click tool to strip AI metadata, inject real phone EXIF (iPhone, Samsung, etc.), and add realistic camera noise. Bypass AI detection on TikTok, Instagram & Xiaohongshu.

## Features

- 🧹 **Strip AI Metadata** — Removes all AI generation markers from EXIF
- 📱 **Real Phone EXIF** — Inject iPhone, Samsung, Xiaomi, OPPO, vivo, HUAWEI camera data
- 🎨 **Real Photo Effects** — Add camera noise, vignette, and color profiles
- 🔒 **100% Private** — All processing in browser, no server upload
- 📦 **Batch Support** — Process multiple images, download as ZIP

## Tech Stack

- Vanilla HTML/CSS/JS
- Canvas API (client-side image processing)
- piexifjs (EXIF injection)
- JSZip (batch download)
- Deployed on Vercel
